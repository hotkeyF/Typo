# Typo.
Sta je Typo?  

Ovo je aplikacija koja sluzi za unaprijedjivanje brzine kucanja na mobilnim uredjajima.  


Plan izrade  

 

-Aplikacija ce biti napravljena u Android Studiu uz pomoc Google Maps API-a
